Mapas de Karnaugh

Es un algoritmo que nos permite obtener a una versión reducida de dos niveles lógicos de cualquier expresión booleana.

Tabla de verdad
A B C F
0 0 0 $F_0$
0 0 1 $F_1$
0 1 0 $F_2$
0 1 1 $F_3$
1 0 0 $F_4$
1 0 1 $F_5$
1 1 0 $F_6$
1 1 1 $F_7$
Mapa de Karnaugh
A\BC 00 01 11 10
0 $F_0$
 $F_1$ $F_3$ $F_2$
1 $F_4$

Trabajo individual

Actividad 9.1: Actividad individual

Plantee el mapa de Karnaugh para un sistema que recibe un número binario de 4 bits y que su salida es 1 para los múltiplos de 2 (no considere el 0 como múltiplo de 2).

Don't cares

En una tabla de verdad o mapa de Karnaugh, un don’t care (x) representa una salida que no es relevante, es decir, puede tomar valor 0 ó 1.

Ejemplo: Uso de don't cares en mapa mapa de Karnaugh

Plantee el mapa de Karnaugh para un sistema que recibe un número binario de 4 bits que representa los meses de enero a diciembre y que su salida es 1 si el mes tiene 31 días.
